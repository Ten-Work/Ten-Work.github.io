# 隨你抽

很高興您喜歡「隨你抽」（之後統一稱為「本應用程式」），本應用程式是輕鬆抽籤，幫助你快速決策的好幫手。

###  應用簡介

隨你抽是一款精心打造，操作流暢又直覺的抽籤工具。
支援多種抽籤方式：選項抽籤、數字抽籤、權重抽籤，甚至密碼產生都能一次搞定。
無論是日常小決定，還是聚會遊戲、幸運抽籤，都能快速得到結果。
免費版即可享用核心功能，升級解鎖更多抽籤組合與便利工具，讓你的決策更加自由無限！

# Pick Anything

Glad that you like "Pick Anything" (hereinafter collectively referred to as "this application"). This application lets you make quick and easy draws for all your decisions.

###  Introduction to the App

Pick Anything is a sleek and intuitive random picker designed for smooth and effortless use.
Supports multiple draw types: item lists, number ranges, weighted choices, and even password generation.
Perfect for daily decisions, party games, or lucky draws—get results in an instant.
Enjoy core features for free, or upgrade to unlock more draw sets and handy tools for unlimited possibilities!
